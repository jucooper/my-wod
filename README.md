# My WOD
A cli app that scrapes the workout from pushjerk.com

## Usage
* `$ my-wod today`
* `$ my-wod tomorrow`
* `$ my-wod yesterday`

## Development
### Setup

1. `$ dep ensure` within the `api/` directory.

### Run

1. `$ go build`
2. `$./my-wod`

## Acknowledgments
